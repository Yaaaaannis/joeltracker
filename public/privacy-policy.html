Politique de Confidentialité - Joel Stream Tracker

Date d'entrée en vigueur : 1 Juin 2024

1. Introduction

Bienvenue dans la politique de confidentialité de l'extension "Joel Stream Tracker". Cette politique explique comment notre extension collecte, utilise et protège vos informations lorsque vous l'utilisez pour suivre les streams de bmsjoel sur Twitch.

Nous nous engageons à respecter votre vie privée et à protéger vos données personnelles. Veuillez lire attentivement cette politique pour comprendre nos pratiques concernant vos données.

2. Données collectées

Notre extension collecte uniquement les données nécessaires à son fonctionnement :

  • Préférences utilisateur (fréquence de vérification, activation des notifications, thème)
  • Informations temporaires sur le statut du stream (titre, jeu, nombre de spectateurs)
  • Horodatage du dernier stream et durée des streams sur les 30 derniers jours

Toutes ces données sont stockées localement sur votre appareil via l'API chrome.storage.local et ne sont pas transmises à nos serveurs.

3. Utilisation des données

Les données collectées sont uniquement utilisées pour :

  • Personnaliser votre expérience utilisateur selon vos préférences
  • Vous notifier lorsque le streamer est en direct
  • Afficher les informations pertinentes sur le stream (titre, jeu, nombre de spectateurs)
  • Calculer et afficher le temps écoulé depuis le dernier stream
  • Calculer et afficher le temps total de stream sur les 30 derniers jours

4. Communication avec des services tiers

Notre extension communique avec l'API Twitch pour récupérer les informations sur le streamer bmsjoel. Ces communications sont nécessaires pour :

  • Vérifier si le streamer est en direct
  • Récupérer les informations du stream (titre, jeu, nombre de spectateurs)
  • Obtenir l'historique des streams pour calculer les statistiques

Aucune donnée personnelle vous concernant n'est partagée avec Twitch au-delà de ce qui est nécessaire pour les requêtes API standard.

5. Autorisations demandées

Notre extension demande les autorisations suivantes :

  • storage : Pour sauvegarder vos préférences et les informations temporaires du stream
  • alarms : Pour vérifier périodiquement si le streamer est en direct
  • notifications : Pour vous alerter lorsque le streamer commence un direct
  • tabs : Pour ouvrir le stream dans un nouvel onglet lorsque vous cliquez sur un lien
  • host_permissions : Pour communiquer avec l'API Twitch (api.twitch.tv et id.twitch.tv)

6. Partage des données

Nous ne vendons, n'échangeons ni ne transférons vos informations personnelles à des tiers. L'extension n'utilise pas de services d'analyse ou de suivi qui pourraient collecter des données sur votre utilisation.

7. Vos droits

Vous pouvez à tout moment :

  • Modifier vos préférences dans les options de l'extension
  • Désinstaller l'extension pour supprimer toutes les données stockées localement

8. Sécurité des données

Toutes les données sont stockées localement sur votre appareil. Nous n'avons pas accès à ces données et elles ne sont pas transmises à nos serveurs.

9. Modifications de cette politique

Nous nous réservons le droit de mettre à jour cette politique de confidentialité à tout moment. Les modifications seront publiées sur cette page avec une nouvelle date d'entrée en vigueur.
